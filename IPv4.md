# 🌍 IPv4 Addressing

## Overview

IPv4 (Internet Protocol Version 4) is the most widely used protocol for identifying devices on a network. Every device connected to a network is assigned a unique IPv4 address for communication.

---

## IPv4 Address Format

An IPv4 address consists of **32 bits** divided into **4 octets**.

Example:

```text
192.168.1.10
```

Each octet ranges from **0 to 255**.

---

## Private IPv4 Address Ranges

| Class | Address Range |
|--------|---------------|
| Class A | 10.0.0.0 – 10.255.255.255 |
| Class B | 172.16.0.0 – 172.31.255.255 |
| Class C | 192.168.0.0 – 192.168.255.255 |

These addresses are used inside private networks and are not directly accessible from the Internet.

---

## Public IP Address

A public IP address is assigned by an Internet Service Provider (ISP) and is used for communication over the Internet.

---

## Loopback Address

```text
127.0.0.1
```

The loopback address is used to test the local machine and is commonly referred to as **localhost**.

---

## Key Points

- IPv4 uses 32-bit addressing.
- It consists of four octets.
- Each octet ranges from 0 to 255.
- Every device on a network requires a unique IP address.

---

## Summary

IPv4 is the foundation of network communication. Understanding IP addressing is essential for networking, system administration, and Cyber Security.
