# 🌐 Subnetting

## Overview

Subnetting is the process of dividing a large network into smaller and more manageable networks called subnets. It improves network performance, enhances security, and makes efficient use of IP addresses.

---

## Why Subnetting is Used

- Reduces network traffic.
- Improves network performance.
- Enhances security by separating networks.
- Makes IP address management easier.

---

## Important Terms

| Term | Description |
|------|-------------|
| IP Address | A unique address assigned to a device on a network. |
| Subnet Mask | Determines the network and host portions of an IP address. |
| Network ID | Identifies the network. |
| Host ID | Identifies a device within the network. |

---

## Common Subnet Masks

| CIDR | Subnet Mask |
|------|-------------|
| /24 | 255.255.255.0 |
| /25 | 255.255.255.128 |
| /26 | 255.255.255.192 |
| /27 | 255.255.255.224 |
| /28 | 255.255.255.240 |

---

## Example

IP Address:

```text
192.168.1.10
```

Subnet Mask:

```text
255.255.255.0
```

Network Address:

```text
192.168.1.0
```

---

## Key Points

- Subnetting divides a network into smaller subnets.
- A subnet mask separates the network and host portions of an IP address.
- CIDR notation is commonly used to represent subnet masks.

---

## Summary

Subnetting is an essential networking concept that helps organize networks efficiently and is widely used in enterprise environments.
