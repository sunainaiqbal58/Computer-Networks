# 🌐 OSI Model

## Overview

The OSI (Open Systems Interconnection) Model is a networking framework that explains how data is transmitted from one device to another over a network. It consists of seven layers, and each layer performs a specific function.

---

## Layers of the OSI Model

| Layer | Name | Main Function |
|--------|------|---------------|
| 7 | Application | Provides network services to end users and applications. |
| 6 | Presentation | Translates, encrypts, and compresses data. |
| 5 | Session | Establishes, manages, and terminates communication sessions. |
| 4 | Transport | Ensures reliable data transfer using TCP or UDP. |
| 3 | Network | Handles logical addressing and routing using IP. |
| 2 | Data Link | Transfers data between devices on the same network using MAC addresses. |
| 1 | Physical | Transmits raw bits through cables, fiber, or wireless media. |

---

## Why the OSI Model is Important

- Helps understand how network communication works.
- Makes troubleshooting easier by separating network functions into layers.
- Provides a standard model for networking technologies.

---

## Key Points

- The OSI Model has **7 layers**.
- Each layer performs a different task.
- Data moves from Layer 7 to Layer 1 before transmission.
- At the receiving end, data moves from Layer 1 back to Layer 7.

---

## Summary

The OSI Model is one of the most important networking concepts. Understanding its layers helps in learning networking, troubleshooting network issues, and building a strong foundation for Cyber Security.
