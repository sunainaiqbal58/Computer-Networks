# 🌐 TCP/IP Model

## Overview

The TCP/IP (Transmission Control Protocol/Internet Protocol) model is the standard communication model used on the Internet. It defines how data is transmitted between devices over a network.

---

## Layers of the TCP/IP Model

| Layer | Function |
|--------|----------|
| Application | Provides network services to applications such as HTTP, FTP, DNS, and SMTP. |
| Transport | Ensures reliable or fast data delivery using TCP or UDP. |
| Internet | Handles logical addressing and routing using IP. |
| Network Access | Manages data transmission over the physical network. |

---

## TCP vs UDP

| TCP | UDP |
|-----|-----|
| Connection-oriented | Connectionless |
| Reliable communication | Faster communication |
| Error checking | No error recovery |
| Used for HTTP, HTTPS, FTP | Used for DNS, VoIP, Streaming |

---

## Common Protocols

- HTTP
- HTTPS
- FTP
- SMTP
- DNS
- SSH

---

## Why TCP/IP is Important

- It is the foundation of modern networking.
- Enables communication between devices on the Internet.
- Supports reliable and efficient data transmission.

---

## Summary

The TCP/IP model consists of four layers and is the most widely used networking model. Understanding TCP/IP is essential for Networking and Cyber Security.
