# 🔌 Common Network Ports

## Overview

Network ports allow different services and applications to communicate over a network. Each service uses a specific port number to send and receive data. Understanding common ports is essential for networking, troubleshooting, and cyber security.

---

## Frequently Used Ports

| Port | Protocol | Service | Purpose                                 |
|------|----------|---------|---------                                |
| 20/21| TCP      | FTP     | File transfer between computers         |
| 22   | TCP      | SSH     | Secure remote login                     |
| 23   | TCP      | Telnet  | Remote access (not secure)              |
| 25   | TCP      | SMTP    | Sending emails                          |
| 53   | TCP/UDP  | DNS     | Resolving domain names into IP addresses|
| 67/68| UDP      | DHCP    | Automatic IP address assignment         |
| 80   | TCP      | HTTP    | Web browsing (unencrypted)              |
| 110  | TCP      | POP3    | Receiving emails                        |
| 143  | TCP      | IMAP    | Accessing emails from a mail server     |
| 443  | TCP      | HTTPS   | Secure web browsing                     |
| 3389 | TCP      | RDP     | Remote Desktop connection               |

---

## Ports Commonly Seen in Cyber Security

### Port 22 – SSH
Used for secure remote access to Linux and Unix systems.

### Port 53 – DNS
Converts domain names (such as google.com) into IP addresses.

### Port 80 – HTTP
Transfers website data without encryption.

### Port 443 – HTTPS
Transfers encrypted web traffic using SSL/TLS.

### Port 3389 – RDP
Allows users to remotely access Windows systems.

---

## Why Learn Port Numbers?

Knowing common ports helps in:

- Network troubleshooting
- Firewall configuration
- Nmap scanning
- Wireshark packet analysis
- Identifying running services
- Understanding cyber security attacks

---

## Summary

Network ports play an important role in communication between devices. A good understanding of common ports is useful for networking, system administration, and cyber security.
